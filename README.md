# Project-1-Loan-Repayment-Prediction-System-
A machine learning project that uses the **Support Vector Machine (SVM)** algorithm to predict loan repayment based on applicant data. It helps financial institutions assess credit risk and make faster, more accurate loan approval decisions.

Overview:
The Loan Repayment Prediction System is a machine learning project that predicts whether a borrower is likely to repay a loan using the Support Vector Machine (SVM) algorithm. The system analyzes applicant information to help financial institutions make informed, data-driven lending decisions and reduce credit risk.

Features:
Predicts loan repayment status using machine learning
Implements the Support Vector Machine (SVM) classification algorithm
Includes data preprocessing and feature scaling
Evaluates model performance using classification metrics
Provides a simple and modular implementation

Technologies Used:
Python
Pandas
NumPy
Scikit-learn 
Google Colab

Project Workflow:
Data Collection
Data Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Feature Scaling
Train-Test Split
Model Training using SVM
Model Evaluation
Prediction

Model Information:
Algorithm: Support Vector Machine (SVM)
Problem Type: Binary Classification

Objective:
The objective of this project is to predict the loan repayment capability of applicants using historical data, enabling financial institutions to minimize credit risk and improve the loan approval process.
